Code Console Basic Pre-populated Example

The editor is placed on the page using <div class="ide"></div>. A <script> runs the following JavaScript:

$(document).ready(function() {
    $(".ide").ide({
        tabs: [
            {
                name: "Example.java",
                text: "public class Example {\n" + 
                        "  public static void main(String[] args) {\n" +
                        "    System.out.println(\"Hello World\");\n  }\n}"
            }, {
                name: "Person.java",
                text: "public class Person {\n\n}"
            }
        ]
    });
});
