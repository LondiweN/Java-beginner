Code Console Basic Theme Example

The editor is placed on the page using <div class="ide"></div>. A <script> runs the following JavaScript:

$(document).ready(function() {
    $(".ide").ide({
        theme: "solarized light",
        tabs: [
            {
                name: "Example.java",
                text: "public class Example {\n" + 
                        "  public static void main(String[] args) {\n" +
                        "    System.out.println(\"Hello World\");\n  }\n}"
            }, {
                name: "Person.java",
                text: "public class Person {\n\n}"
            }
        ]
    });
});

http://codemirror.net/ is used for the in-browser code editor. To view the available themes visit http://codemirror.net/demo/theme.html. The toolbar and button tabs can be changed using jQuery UI themes. View the source of this page for the complete listing of CSS and JavaScript files to load.
